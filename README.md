**Dizi:**  
[16, 21, 11, 8, 12, 22]

### Aşamalar

**Bölme (Divide):**

[16, 21, 11, 8, 12, 22]  
[16, 21, 11] – [8, 12, 22]  
[16] – [21, 11] – [8] – [12, 22]  
[16] – [21] – [11] – [8] – [12] – [22]

Birleştirme (Merge):

[21] + [11] → [11, 21]  
[16] + [11, 21] → [11, 16, 21]  

[12] + [22] → [12, 22]  
[8] + [12, 22] → [8, 12, 22]  

**Son Birleştirme:**  
[11, 16, 21] + [8, 12, 22] →  
[8, 11, 12, 16, 21, 22]

[8, 11, 12, 16, 21, 22]
